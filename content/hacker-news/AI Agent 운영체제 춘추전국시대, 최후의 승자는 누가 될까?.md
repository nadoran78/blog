
최근 Hermes Agent를 살펴보다가 재미있는 생각이 들었다.

요즘 AI 업계는

> AI Agent 전쟁

이라기보다

> AI Agent 운영체제 전쟁

에 더 가까워 보인다.

## Agent와 Agent 운영체제는 다르다

처음에는 나도 헷갈렸다.

Claude Code, Codex, Hermes, OpenClaw가 전부 비슷한 부류처럼 보였기 때문이다.

하지만 조금 살펴보니 서로 다른 계층에 있는 프로젝트들이었다.

### Agent

이미 상당히 완성된 형태의 Agent

- Claude Code
    
- Codex
    
- Devin
    

이들은 실제로

```text
파일 읽기
코드 수정
명령 실행
테스트 수행
```

같은 작업을 수행한다.

즉,

> "일을 하는 존재"

에 가깝다.

---

반면

### Agent 운영체제

- Hermes
    
- OpenClaw
    
- Goose
    
- Jaiph
    

이들은 조금 다르다.

이들의 관심사는

```text
기억(Memory)

Skill

자동화

스케줄링

멀티 에이전트

메신저 연동
```

이다.

즉,

> Agent를 어떻게 관리하고 운영할 것인가?

에 초점이 맞춰져 있다.

## 마치 웹 브라우저 전쟁 같기도 하다

예전 인터넷 초창기에는

```text
Netscape

Internet Explorer

Opera

Firefox
```

가 경쟁했다.

당시에는

> 최종 승자가 누가 될지 아무도 몰랐다.

결국 Chrome이 시장을 장악했지만, 당시에는 전혀 당연한 결과가 아니었다.

지금의 Agent 운영체제 시장도 비슷하게 느껴진다.

## 각자의 철학도 조금씩 다르다

### Hermes

Hermes는

> 나를 기억하는 개인 비서

에 가깝다.

장기 기억과 Skill 자동 생성에 집중한다.

문제를 해결하면 SKILL.md를 생성하고 다음부터 재사용한다.

---

### OpenClaw

OpenClaw는

> Agent 허브

에 가깝다.

Telegram, Discord, Slack 등 다양한 채널을 연결하고 에이전트를 중심으로 소통하는 데 초점이 있다.

---

### Goose

Goose는

> 개발자를 위한 범용 Agent Runtime

느낌이다.

상대적으로 가볍고 실용적인 방향을 추구한다.

---

### Jaiph

Jaiph는 조금 독특하다.

> Agent를 프로그래밍 언어처럼 다룬다.

작업 큐를 기반으로 에이전트가 지속적으로 작업을 수행하도록 설계되어 있다.

## 그런데 정말 중요한 것은 따로 있는 것 같다

재미있는 점은 프로젝트 이름보다 개념이 더 중요해 보인다는 것이다.

예를 들어

```text
Memory

Skill

MCP

Automation

Sub-Agent
```

같은 개념들 말이다.

지금은 Hermes가 이 기능을 강조하고,

OpenClaw가 저 기능을 강조하지만,

몇 년 뒤에는 대부분의 Agent 운영체제가 비슷한 기능을 제공하게 될 수도 있다.

마치 오늘날의 브라우저들이 모두 탭, 북마크, 확장 프로그램을 지원하는 것처럼 말이다.

## 나는 아직 관망 중이다

솔직히 말하면 지금 당장 어떤 Agent 운영체제에 정착할 생각은 없다.

너무 많은 프로젝트가 동시에 등장하고 있다.

어떤 프로젝트는 1년 뒤 사라질 수도 있고,

어떤 프로젝트는 갑자기 시장을 장악할 수도 있다.

그래서 지금은

```text
옥석가리기 진행 중
```

상태라고 생각한다.

물론

- MCP
    
- Memory
    
- Skill
    

같은 개념은 매우 흥미롭다.

하지만 특정 Agent 운영체제를 지금 선택해야 할 이유는 아직 잘 모르겠다.

## 최후의 승자는 누가 될까?

솔직히 전혀 모르겠다.

Hermes가 될 수도 있고,

OpenClaw가 될 수도 있고,

지금은 존재감이 크지 않은 다른 프로젝트가 될 수도 있다.

심지어 몇 년 뒤에는 Anthropic이나 OpenAI가 공식 Agent 운영체제를 내놓을 수도 있다.

다만 한 가지는 확실하다.

지금은

> AI Agent 시대의 시작

이라기보다

> AI Agent 운영체제 춘추전국시대의 시작

처럼 보인다.

그리고 나는 일단 관전자석에 앉아 팝콘을 먹으며 지켜볼 생각이다.

옥석가리기가 끝나고 나면 그때 진지하게 써봐도 늦지 않을 것 같다.
